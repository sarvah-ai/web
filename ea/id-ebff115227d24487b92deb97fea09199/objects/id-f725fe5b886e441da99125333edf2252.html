Narrator Agent ()
●Goal: To select the optimal narrative strategy (Playbook) and compose a multi-faceted, coherent, and engaging answer that is perfectly tailored to the user's quest. ●Enriched Responsibilities: ○Context-Aware Playbook Selection: The choice of Playbook is highly dynamic. If the Verification Agent flags a scholarly debate, the Narrator might switch from a simple "Iconography Playbook" to a "Contested Interpretations Playbook." ○Multi-Part Response Composition: It constructs modular responses that can include a poetic one-line summary, a clear evidence block ("Here's why we think so..."), links to primary sources, and interactive elements for further exploration. ○Epistemic Hedging: It masterfully uses language to convey uncertainty and confidence. It avoids definitive statements where the evidence is weak, using phrases like "Art historians suggest..." or "One interpretation is..." ○Dynamic Follow-ups: Based on the complexity of the topic, it can proactively suggest follow-up questions to the user, guiding them deeper into the subject matter.

Interpreter Agent ()
Goal: To translate ambiguous, multi-modal user input into a structured, machine-readable IntentBundle that captures the full context and nuance of the query. ●Enriched Responsibilities: ○Semantic Parsing: Goes beyond keywords to analyze the grammatical structure and semantic relationships in the user's text to infer the core question. ○Multi-Modal Fusion: Correlates textual queries with data extracted from images. It identifies not just objects (e.g., "a dog") but also compositional relationships (e.g., "a dog at the feet of a couple") that are critical for context. ○Profile-Based Contextualization: Uses the user's profile (familiarity level, language) to infer the required depth and complexity of the answer. For a novice, it might prioritize a clear, concise explanation; for an expert, it might highlight scholarly debates. ○Intent Disambiguation: Actively identifies ambiguity (e.g., a symbol with multiple meanings) and flags it in the AmbiguityTokens output, preparing the system to either ask for clarification or present all possible interpretations.
